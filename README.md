# PyNeon

PyNeon is a light-weight package to work with Neon (Pupil Labs) multi-modal eye-tracking data.

## Documentation

[PyNeon Documentation](https://ncc-brain.github.io/PyNeon/) contains public API and tutorials as Jupyter notebooks.

## Citing PyNeon

If you use PyNeon in your research, please cite the
[accompanying paper](https://osf.io/preprints/psyarxiv/y5jmg)
as follows:

```bibtex

    title={PyNeon: a Python package for the analysis of Neon multimodal mobile eye-tracking data},
```
